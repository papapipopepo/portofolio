## Stack

Python 3, MySQL (MySQL Connector), Matplotlib

## Fitur

- Dual role: Admin / Borrower
- Auto fine calculation untuk late return
- 6 laporan: avg borrowing duration, publication year, status total, top 3 borrower/book, fine analytics
- 5 jenis visualisasi: pie, histogram, bar
- Auto-generated borrowing ID (mulai 1001)
- Real-time stock management
